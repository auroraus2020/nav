一、打开网站的第一眼

在浏览器打开 https://nav.auroraus.cn，你会看到：

  • 大搜索框 — 页面正中间，输入关键词就能搜
  • 搜索引擎选择栏 — 搜索框下面一排按钮，每个按钮代表一个搜索引擎
  • 常用站点 — 你点得最多的网站，会自动出现在搜索框下面
  • 分类导航卡片 — 按"综合导航""学术检索""AI大模型"等分类排列的网站卡片
  • 左侧侧边栏（电脑）或 底部Tab栏（手机）— 快速跳转到不同分类
  • 右侧快速工具栏（电脑）— 一排竖着的图标，一键打开微信传输、计算器、画图等常用网站
🖱️ 打开一个网站：直接点任意卡片，网站在新标签页打开，本页保持不变。
📂 收起/展开一个分类：点分类标题（比如"学术检索"），下面所有卡片会收起或展开。鼠标点一下就行。
🔍 搜索本页的网站：在搜索框输入关键词（比如输入"知乎"），页面会只保留名称或描述包含"知乎"的卡片。按 Esc 键或手动清空搜索框恢复到全部显示。
💡 常用站点区：你每点一个网站卡片，系统就记一次数。点得最多的几个会自动出现在"常用站点"区。不需要你特意收藏。

二、搜索引擎怎么用

搜索框不只是用来过滤导航卡片的——它更是一个能切换多种搜索引擎的搜索聚合器。

🔀 切换搜索引擎（三种方式）

方式一：鼠标点 — 点搜索框下面你想用的引擎就行了，选中的会高亮。
方式二：按 Ctrl 键 — 单独按一下 Ctrl（不要同时按别的），引擎会按搜索框下面的显示顺序挨个切换。只会在你开启的那些引擎之间循环。
方式三：按 F1 ~ F12 — F1 选搜索框下面第一个可用引擎，F2 选第二个……也是按显示顺序。

🔎 执行搜索

在搜索框里打好你要搜的关键词 → 按 回车键 或点旁边的"搜索"按钮 → 浏览器开新标签页用你选的引擎搜索。

网页往下滚一段距离后，页面顶部会浮出一个悬浮搜索条，显示当前选中的搜索引擎名字，随时可以搜，不用滚回顶部。

💡 怎么让某个引擎出现/消失：去设置面板 → "搜索引擎显示"，点它对应的圆角按钮。蓝色 = 显示，灰色 = 隐藏。新导入的自定义引擎默认是隐藏的，记得去打开。

自定义工具

你可以在配置编辑器中添加任意数量的自定义快捷工具，每个工具可以设置：

  • 图标 — 任意emoji表情
  • 名称 — 鼠标悬停时显示的提示文字
  • 网址 — 点击时打开的链接
  • 启用/禁用 — 控制某个工具是否显示
怎么添加自定义工具：打开设置 → 点 🖊️ 编辑器 → 切换到 🔗 快捷工具 标签页 → 在这里管理你的自定义工具 → 点 💾 保存。

全局开关

在编辑器的快捷工具页面，可以用全局开关一键启用或禁用整个右侧工具栏。禁用后所有图标都会消失。

💡 为什么不在主页设置里管理？快捷工具的自定义配置项较多（图标、名称、URL、启用状态），放在编辑器里用表格管理更方便。导出/导入配置时会自动包含自定义工具。

📋 待办事项

在右侧工具栏最下方有一个📋图标，点击后会弹出一个半透明悬浮窗，可以用它来管理你的待办事项。

功能说明

  • 添加待办 — 在输入框中输入内容，按回车或点"添加"按钮即可新增待办
  • 切换状态 — 点击待办项前面的方框，可以在"未完成"和"已完成"之间切换
  • 筛选查看 — 顶部有"全部""未完成""已完成"三个标签页，方便分类查看
  • 删除待办 — 鼠标悬停在待办项上，右侧会出现 × 删除按钮
  • 重要性等级 — 每条待办支持设置三个等级：● 一般 / ● 重要 / ● 紧急，分别以蓝、橙、红三色圆点标记，点击圆点即可循环切换
  • 截止日期与倒数日 — 可为待办设置截止日期，待办项右侧会显示距离截止日的倒数天数（如"剩3天""已过期2天"），过期任务以红色高亮提醒
  • 自动清理 — 已完成超过2天的待办事项会被自动清除，不再显示
  • 拖拽移动 — 按住悬浮窗标题栏，可以把它拖到页面上任意位置
  • 关闭窗口 — 点击标题栏右侧的 ✕ 按钮即可关闭（不会自动消失，只有手动关闭）

导入/导出

悬浮窗标题栏有📤导出和📥导入按钮：

  • 📤 导出 — 将所有待办事项导出为 .json 文件，方便备份或迁移
  • 📥 导入 — 选择之前导出的 .json 文件，恢复待办数据
💡 数据存储：待办数据保存在浏览器的本地存储中，不会上传到任何服务器。清除浏览器缓存或更换浏览器会导致数据丢失，建议定期导出备份。

📝 记事本

在右侧工具栏中有一个📝图标，点击后会弹出一个Markdown 记事本悬浮窗，可以随时随地记录笔记。

功能说明

  • 编辑 / 预览双模式 — 顶部有"编辑"和"预览"两个标签页切换。编辑模式下自由书写 Markdown 文本，预览模式下实时渲染为格式化的富文本
  • 📥 导入 .md 文件 — 点击标题栏的 📥 按钮，可从本地选择 .md 或 .txt 文件导入到编辑区
  • 📤 导出 .md 文件 — 点击标题栏的 📤 按钮，将当前笔记内容导出为 .md 文件下载保存
  • 🗑️ 清空内容 — 点击标题栏的 🗑️ 按钮可一键清空所有笔记内容（操作前有确认提示）
  • 拖拽移动 — 按住悬浮窗标题栏，可以将记事本拖到页面上任意位置
  • 自由调整大小 — 拖拽悬浮窗右下角的三角形手柄，可以自由拉伸窗口的宽度和高度
💡 数据存储：笔记内容保存在浏览器本地存储中，不会上传到任何服务器。在配置编辑器中导出/导入全局配置时会一并包含笔记数据。建议定期导出 .md 文件备份重要笔记。

🔁 怎么让改动生效：
第1步：在编辑器里改完 → 点 💾 保存
第2步：回到主页 https://nav.auroraus.cn/ → 刷新页面

💡 高级功能：编辑器也支持 📤 导出JSON 和 📂 导入JSON，可以把你选好的配置导出给其他人一键导入。

十、编辑器怎么和主页同步？

很多用户问：这是静态页面，编辑器改的东西怎么跑到主页上去的？

答：通过浏览器的 localStorage（本地存储）。

  • 编辑器保存时 → 把配置写入浏览器的 localStorage
  • 主页加载时 → 从浏览器 localStorage 读取配置并渲染
  • 因为是同一个域名（nav.auroraus.cn），两个页面共享同一份 localStorage
  • 不需要服务器、不需要数据库、不需要API — 纯前端就搞定了
操作流程：编辑器改 → 💾保存 → 打开主页刷新 → 生效 ✅
⚠️ 注意：如果你清除了浏览器的"网站数据"，配置会丢失。建议定期点 📤 导出JSON 做备份。

十一、通过JSON给已有分类增删站点

不需要改网页代码！只要在 JSON 配置文件里加两个字段：

✚ 往已有分类里添加站点

在 JSON 文件里加一个 addSites 字段，格式是 { "分类ID": [站点信息...] }。

比如，在"综合导航"分类里加两个新网站，JSON 文件这样写：

{
  "addSites": {
    "cat-general": [
      {"name":"V2EX","url":"https://www.v2ex.com","desc":"创意工作者社区","icon":"V","iconColor":"blue"},
      {"name":"Product Hunt","url":"https://www.producthunt.com","desc":"新产品发现平台","icon":"P","iconColor":"orange"}
    ]
  }
}

导入后，这两个新卡片就会直接出现在"综合导航"分类的末尾。再次导入同一个 JSON 不会重复添加。

✖ 从已有分类里移除站点

在 JSON 文件里加 removeSites 字段，填上分类ID和要移除的站点名称列表。

比如，把"技术交流"分类里的"V2EX"和"掘金"两个站点隐藏掉：

{
  "removeSites": {
    "cat-tech": ["V2EX", "掘金"]
  }
}

站点名称必须和页面上显示的完全一致（区分大小写）。

💡 addSites 和 removeSites 可以和 categories、engines、settings 写在同一个 JSON 文件里，一次导入全部生效。导出时这些信息也会一并导出。

十二、导出本地壁纸（含 base64）

如果你用了本地图片作为壁纸，默认情况下不会导出，保护你的隐私。

但如果你想把这个壁纸一起分享给朋友，可以这样做：

第1步：设置面板 → 壁纸来源 → 选"本地图片壁纸"
第2步：下面会冒出来一个复选框 ☑️ "导出JSON时包含本地壁纸（base64格式，放在文件末尾）"
第3步：勾选它
第4步：点 📤 导出配置

导出的 JSON 文件末尾会多一个 _base64Wallpaper 字段，包含壁纸的 base64 编码数据。这个字段在最末尾，不影响你查看和修改前面其他配置内容。

朋友导入这个 JSON 后，本地壁纸也会自动恢复。

⚠️ base64 数据可能非常大（几MB），文件会比较大。建议只在需要分享壁纸时才勾选。

十六、常见问题

❓ 编辑器改完保存了，主页刷新没变化？

试试硬刷新（Ctrl + F5），或者先关掉主页标签页再重新打开。如果还不生效，检查一下是不是在浏览器隐身模式下操作（隐身窗口的 localStorage 是隔离的）。

❓ 导入JSON后新加的搜索引擎看不到？

自定义搜索引擎默认是隐藏的。去设置 → 搜索引擎显示，找到它的灰色按钮点一下变蓝就出来了。

❓ 导入后自定义分类消失？

去设置 → 分类显示，看它的按钮是不是灰色的。点蓝就显示。

❓ 怎么只保留我需要的几个分类？

设置 → 分类显示 → 把不要的分类按钮点成灰色，页面和侧边栏就都看不到了。

❓ 我的设置存在哪？

存在浏览器的 localStorage 里。只要不清浏览器数据就一直保留。

❓ 怎么把导航页放到手机桌面？

用手机浏览器打开 https://nav.auroraus.cn → 浏览器菜单 → "添加到主屏幕"/"添加到桌面"。

❓ F1~F12 不灵？

同时按住 Fn，或者在BIOS里把功能键设为标准F1-F12。
