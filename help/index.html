一、基本使用

打开 AuroraUs 导航页面，你会看到：

  • 顶部搜索区 — 一个大搜索框和下方一整排搜索引擎按钮
  • 常用站点区 — 搜索框下方，显示你点击最多的几个网站
  • 分类导航区 — 按类别（综合导航、学术检索、AI大模型等）组织的网站卡片
  • 左侧侧边栏（电脑端）或 底部Tab栏（手机端）— 快速跳转到每个分类
🖱️ 打开一个网站：点击任意一张卡片，网站会在新标签页中打开。
📂 折叠/展开分类：点击分类标题（如"学术检索"），该分类下的所有卡片会收起或展开。
🔍 搜索导航网站：在搜索框输入关键词（比如"知乎"），页面只显示名称或描述中包含该词的卡片。想恢复全部显示，按 Esc 或清空搜索框即可。
💡 常用站点自动记录：你点击某个网站卡片时，系统会自动记录点击次数。点击最多的前几个网站会出现在搜索框下方的"常用站点"区域，方便快速访问。

二、搜索引擎使用

搜索框里的内容不仅仅用来过滤导航卡片——它更是一个聚合搜索引擎，支持在多个搜索引擎之间快速切换并搜索。

🔀 切换搜索引擎

方法一：鼠标点击 — 直接点击搜索框下方的引擎名称（如"🔵 必应"、"🔴 百度"），当前选中的引擎会高亮。
方法二：键盘按 Ctrl（单独按一下） — 按搜索框下方显示的顺序，依次切换到下一个可见的搜索引擎。
方法三：功能键 F1 ~ F12 — 按 F1 选第1个可见引擎，F2 选第2个……依此类推（按搜索框下方的显示顺序）。

🔎 执行搜索

在搜索框输入关键词后，按回车键或点击右边"搜索"按钮，会在新标签页用当前选中的引擎进行搜索。

页面往下滚动后，顶部会出现一个悬浮搜索条，显示当前搜索引擎的名称，方便随时搜索，不用滚回顶部。

💡 引擎显示/隐藏：默认有些搜索引擎是隐藏的（如抖音、B站等）。在设置面板的"搜索引擎显示"里点击对应的 chip 可以随时开关。

三、键盘快捷键

快捷键功能说明
Ctrl + K聚焦搜索框光标跳到搜索框
Alt + S聚焦并全选搜索框方便快速输入新内容
Esc清空搜索框恢复显示所有导航卡片
Ctrl（单独按）切换到下一个可见引擎按搜索框下方的显示顺序依次切换
F1 ~ F12快速选择搜索引擎F1=第1个可见引擎，F2=第2个……
⚠️ 注意：F1~F12 在某些浏览器中可能被系统占用。如果无效，请尝试同时按住 Fn 键。

四、设置面板详解

🔧 打开设置：点击页面左上角的 ⚙️ 齿轮图标（电脑端在左上角，手机端也在左上角）。设置面板会从左侧滑出。

🖼️ 壁纸来源（4种模式）

  • 在线风景壁纸 — 每天从本地风景池随机选取一张。
  • 默认壁纸 — 使用导航页自带的默认背景图。
  • 自定义网络壁纸 — 贴入网络图片URL。支持 JPG / PNG / GIF 动图。
  • 本地图片壁纸 — 从你的电脑/手机选择图片文件，存在浏览器中。

选择"本地图片壁纸"后，点击 📁 选择本地图片 按钮即可从本机选图。

🌓 主题模式

切换浅色模式和深色模式。深色模式适合夜间使用，减少眼睛疲劳。

📐 导航折叠

一键展开或折叠全部导航分类。折叠后页面只显示搜索区域，更简洁。

🔍 搜索引擎显示

点击每个引擎的 chip 来开关其是否显示在搜索框下方。蓝色 = 显示，灰色 = 隐藏。

⭐ 常用站点

开关常用站点区域的显示，以及设置显示数量（1~12个）。

📂 分类显示

点击每个分类的 chip 来开关该分类是否在主页面显示。

📥 导入/导出自定义

📂 导入 — 选择 JSON 配置文件，一键导入自定义分类、导航站点和搜索引擎。

📤 导出 — 将当前配置（含设置）下载为 JSON 文件，可以分享给朋友。

🔄 恢复默认 — 清除所有自定义数据和设置，恢复到初始状态。

五、壁纸设置

在线风景壁纸

导航页会自动扫描 assets/ 文件夹中以 wp_ 开头的图片文件。支持的格式：

  • assets/wp_01.jpg ··· assets/wp_99.jpg（传统风景壁纸）
  • assets/wp_01.gif ··· assets/wp_99.gif（动图壁纸，支持动画效果）

把下载的风景图按编号命名放进 assets/ 文件夹，刷新页面就会自动收录。

每天会随机选一张，同一张图当天不会变。

💡 动图壁纸：放入 wp_13.gif 等文件，扫描器会先尝试 .jpg，找不到再试 .gif。

自定义网络壁纸

在设置中切换到"自定义网络壁纸"，然后输入图片的网络地址（URL），按回车或点击输入框外就生效了。支持GIF动画。

这个URL会随配置 JSON 导出，所以收到你配置文件的朋友也能看到同一张壁纸。

本地图片壁纸

点击 📁 选择本地图片，从你的电脑选一张图。图片以 base64 编码存储在浏览器中，不会随 JSON 导出（导出时壁纸地址留空），保护你的隐私。

⚠️ 注意：本地图片存储大小有限制（通常约5MB）。太大的图片可能无法存储。

六、JSON 导入/导出配置

📥 导入配置步骤

第1步：打开设置面板（⚙️齿轮）。
第2步：点击 📂 导入配置文件 按钮。
第3步：在弹出的文件选择框中，选择你的 .json 配置文件。
第4步：系统提示"导入成功"，页面自动刷新，新配置生效！

📤 导出配置步骤

第1步：打开设置面板。
第2步：点击 📤 导出配置 按钮。
第3步：浏览器自动下载一个 auroraus_config_日期.json 文件。
第4步：把这个文件发给朋友，他们导入后就有和你一样的导航配置！
💡 适用场景：
  • 换了电脑或浏览器，导出配置再导入，一键恢复
  • 团队共享一套导航配置（内部工具、常用资源）
  • 备份自己的配置，防止浏览器数据被清理

🔄 恢复默认

点击后弹出确认框，确认后清除所有自定义数据（包括设置、壁纸、分类显示偏好等），页面恢复到出厂状态。

⚠️ 此操作不可撤销！建议先导出配置备份再执行。

七、JSON 配置格式（完整参考）

完整 JSON 结构

{
  "categories": [                         // 自定义分类（可选）
    {
      "id": "cat-mytools",               // 唯一ID，必填，建议 "cat-xxx" 格式
      "name": "我的工具",                 // 分类名称，必填
      "icon": "assets/icon_tool.jpg",    // 图标路径，可选，可用 assets/ 下的图
      "sites": [                         // 该分类下的站点列表
        {
          "name": "GitHub",             // 站点名称，必填
          "url": "https://github.com",  // 站点链接，必填（完整URL）
          "desc": "代码托管平台",        // 简短描述，可选
          "icon": "G",                  // 图标显示字母，可选，默认取name首字母
          "iconColor": "blue"           // 图标颜色，可选，见下方颜色表
        }
      ]
    }
  ],
  "engines": [                            // 自定义搜索引擎（可选）
    {
      "id": "npmjs",                     // 引擎唯一标识，必填（英文、数字）
      "name": "npm搜索",                  // 引擎名称，必填
      "url": "https://npmjs.com/search?q=", // 搜索URL，必填，用 ?q= 或 ?keyword= 等
      "label": "📦 npm"                  // 在引擎列表中显示的标签，可选（含emoji）
    }
  ],
  "settings": {                           // 页面设置（所有字段都可选，不传则用默认值）
    "theme": "dark",                      // 主题：light（浅色）| dark（深色）
    "wallpaper": "custom",                // 壁纸：bing | default | custom
    "customWallpaper": "https://图片地址", // 自定义壁纸URL（wallpaper=custom时有效）
    "freqVisible": true,                 // 常用站点：true=显示 | false=隐藏
    "freqCount": 8,                      // 常用站点数量：1-12
    "navFold": "show",                   // 导航折叠：show=展开 | hide=折叠
    "hiddenCategories": {                // 分类显隐控制
      "cat-general": false,              // false=显示，true=隐藏
      "cat-academic": false
    },
    "hiddenEngines": {                   // 搜索引擎显隐控制
      "bing": false,                     // false=显示，true=隐藏
      "baidu": false
    }
  }
}

九、常见问题（FAQ）

❓ 导入配置后，自定义分类为什么看不到？

请检查设置面板 → 分类显示，确保自定义分类的 chip 是蓝色（显示状态）。如果是灰色，点击一下即可开启。

❓ 导入的搜索引擎为什么选不了？

自定义搜索引擎默认是隐藏的。请在设置面板 → 搜索引擎显示中，点击对应的 chip 把它变成蓝色（显示状态），然后就能在搜索框下方看到并使用了。

❓ 我的设置会保存在哪里？

所有设置（主题、壁纸、引擎显隐、分类显隐、常用站点等）都保存在浏览器的 localStorage 中。只要你不在浏览器中清除网站数据，设置会一直保留。

❓ 换了浏览器或电脑，设置还在吗？

不会自动同步。建议定期使用"📤 导出配置"功能备份，换设备后通过"📂 导入配置文件"恢复。

❓ 动图壁纸不流畅怎么办？

GIF 文件过大可能导致加载缓慢和卡顿。建议使用压缩过的 GIF（5MB以内），或使用 Wallpaper Engine 等工具生成的高质量动图。

❓ F1~F12 快捷键没反应？

部分笔记本电脑的 F1~F12 默认是功能键（亮度、音量等），需要同时按住 Fn 键使用。
或者在BIOS/系统设置中将功能键行为改为"标准F1-F12"。

❓ 怎样只保留我需要的分类？

设置面板 → 分类显示 → 把不想看到的分类 chip 点灰即可。被隐藏的分类在侧边栏和移动端Tab中也会消失。
